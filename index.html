Keyboard Shortcuts

Navigating the Architecture Map

Tab
Jump to the Architecture Map (then use arrow keys)
↑ / ↓
Move between cards
→
Expand subtree, or move into first child
←
Collapse subtree, or move up to parent
Home / End
Jump to first or last card

Moving Cards

M
Pick up the focused card to move it
↑ / ↓
Cycle through available drop positions
Enter / Space
Confirm and drop the card
Esc
Cancel move, return card to original position

Card Actions

E
Edit the title of the focused card
B
Return card to the Element Box
Del
Delete card (children return to Element Box)
Del / Backspace (in Element Box)
Delete focused Element Box card
Shift+Enter (in Element Box input)
Add elements from the text area

Global

Ctrl+Z
Undo last action
Ctrl+Y
Redo
C
Collapse or expand the Element Box sidebar
?
Show or hide this shortcuts panel
